# 結果報告テンプレート

全タスクの結果を集約して以下の形式でユーザーに報告する。

## テンプレート

```markdown
## 並列タスク実行結果

**実行方式:** 並列 agent / 複数 agent
**タスク数:** N 件
**成功:** X 件 / **失敗:** Y 件

---

### タスク 1: <タスク名> ✅

**結果:**
<結果の要約>

**変更ファイル:** (あれば)

- path/to/file1 (+10, -5)

---

### タスク <番号>: <タスク名> ❌

**エラー:**
<失敗の原因>

**推奨対応:**
<手動対応が必要な場合の指示>
```

## 依存関係がある場合の実行順序テンプレート

```markdown
## 実行順序

**フェーズ 1 (並列):** タスク 1, タスク 3
**フェーズ 2 (タスク 1 完了後):** タスク 2
**フェーズ 3 (全完了後):** タスク 4
```
