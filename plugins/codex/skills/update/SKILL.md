# AGENTS.md 更新ワークフロー

既存の AGENTS.md ファイルを精査し、最適化する。

## 重要な原則

1. **冗長コンテンツを削除** - Codex が既知の一般的なベストプラクティスは含めない
2. **必須セクションを確認** - 日本語スタイリング、コード参照、技術調査優先順位
3. **肥大化を防止** - 目標 500-1,500 words、最大 3,000 words
4. **ユーザー承認を得てから適用**

## 作業開始前の準備

**必須:** 作業開始時に `update_plan` で実行ステップを整理し、以下のステップを登録する:

```
update_plan({
  plan: [
    { step: "AGENTS.md の読み込み", status: "pending" },
    { step: "必須セクションの確認", status: "pending" },
    { step: "冗長性の検出", status: "pending" },
    { step: "最適化提案の作成", status: "pending" },
    { step: "ユーザー承認", status: "pending" },
    { step: "変更の適用", status: "pending" },
    { step: "完了報告", status: "pending" },
  ]
})
```

各ステップの開始時に `update_plan` で `in_progress` にし、完了時に `completed` に更新する。

## 実行手順

### 1. 対象ファイルの決定

- 引数が指定された場合: `$1/AGENTS.md` または `$1`
- 引数がない場合: 現在のディレクトリの `AGENTS.md`

### 2. AGENTS.md の読み込み

```bash
# ファイルの存在確認
ls -la AGENTS.md 2>/dev/null
```

Read ツールで内容を読み込み、文字数を計測。

### 3. 必須セクションの確認

以下が含まれているか確認し、不足があれば追加提案:

- 日本語使用時のスタイリング
- コード参照 (`Grep`、`Read`、`Glob` の利用方針)
- 技術調査優先順位

### 4. 冗長性の検出

以下に該当する内容を特定:

| カテゴリ                             | 判断基準         |
| ------------------------------------ | ---------------- |
| 一般的なベストプラクティス           | Codex は既知     |
| 標準的なコーディング規約             | Codex は既知     |
| 言語・フレームワークの基本的な使い方 | Codex は既知     |
| IDE やエディタの使い方               | Codex は既知     |
| 一般的な Git ワークフロー            | Codex は既知     |
| 繰り返し・重複する内容               | 1 箇所に統合可能 |
| 使用されていない古い情報             | 削除対象         |

### 5. 最適化提案の作成

削除・統合すべき内容をリストアップ:

- 各項目について削除理由を説明
- 統合可能な項目を提案
- 追加すべき必須セクションを提示

### 6. ユーザー承認

変更内容を表示し、ユーザーへの短い確認 で承認を得る:

```
## 最適化提案

### 削除対象
- [項目1]: [理由]
- [項目2]: [理由]

### 追加対象
- [必須セクション]: [不足している場合]

### 変更後の文字数
- 現在: X words
- 変更後: Y words (Z% 削減)

この変更を適用しますか？
```

### 7. 変更の適用

承認後、Edit ツールで変更を適用。

### 8. 完了報告

```
## AGENTS.md 更新完了

- **ファイル:** <path>
- **変更前:** X words
- **変更後:** Y words
- **削減率:** Z%
```

## 保持対象

以下は削除しない:

- プロジェクト固有のワークフロー
- カスタムスクリプトやツールの使い方
- ドメイン固有の規約
- チーム固有のルール
- アーキテクチャ決定事項

## エラーハンドリング

### AGENTS.md が存在しない場合

```
AGENTS.md が見つかりません。`codex:init` で新規作成してください。
```

### 変更が不要な場合

```
AGENTS.md は既に最適化されています。変更は不要です。
```
