# Git Plugin

Codex 向けの Git / GitHub ワークフロー plugin です。

## Skills

- `git:commit`
- `git:japanese-text-style`
- `git:pr-ci`
- `git:pr-create`
- `git:pr-explain`
- `git:pr-fix`
- `git:pr-review`
- `git:pr-status`
- `git:pr-update`
- `git:pr-watch`
- `git:review`
- `git:wt`

## Notes

- review、PR、CI、worktree などの Git / GitHub workflow を skill ごとに分けています。
- コミットや push は、repo の `AGENTS.md` で確認必須にされている場合、その境界を優先します。
